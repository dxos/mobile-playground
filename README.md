# mobile-playground

To start locally:

```bash
yarn && yarn start
```

To start ios:

```bash
yarn && yarn start:ios
```

In XCode:

First, go to project's Signing & Capabilities, and add a team.

![signing](images/singing.png)

Then, select the team, and change Bundle Indentifier to a custom string.
